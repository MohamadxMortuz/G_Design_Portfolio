# Graphics Portfolio - Mohamad Mortuz

A modern, dark-themed portfolio website showcasing graphic design, 3D modeling, and visual communication work. Built with React, TypeScript, and Tailwind CSS.

## About

This portfolio highlights my expertise in:
- **3D Design**: Jewelry and product modeling using Fusion 360 and Blender
- **Visual Communication**: Poster design and infographics with Adobe Illustrator
- **Branding**: Color boards, mood boards, and brand identity development
- **3D Rendering**: Realistic product visualizations and character design
